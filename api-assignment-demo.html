Technical Details

Architecture: The system is built with a modular approach that extends the existing AI infrastructure without breaking changes.

Data Flow:

  1. User configures API assignments via ChatSettingsPanel
  2. Settings instantly propagate to ChatBotModule
  3. All chat interactions route through chatRouter.js
  4. Router selects appropriate API based on role and assignments
  5. Fallback handling ensures reliable service
  6. GA4 events track usage and performance

Compatibility: Fully compatible with existing AIChatBox.jsx and ai.js utilities. No modifications to existing layout or SEO implementation required.
